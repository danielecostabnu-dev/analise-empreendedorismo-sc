# analise-empreendedorismo-sc
Projeto de análise exploratória de dados sobre empreendedorismo em Santa Catarina.
## Principais Insights

A partir da análise exploratória dos dados foi possível observar alguns pontos interessantes:

- Algumas cidades apresentam maior concentração de empresas em determinados setores econômicos.
- O setor industrial aparece com forte presença em cidades como Joinville e Jaraguá do Sul.
- O setor de tecnologia possui destaque em cidades como Florianópolis e São José.
- O setor de turismo apresenta relevância em cidades litorâneas como Balneário Camboriú.

Essas informações podem ajudar a compreender melhor a distribuição das atividades econômicas no estado de Santa Catarina.
